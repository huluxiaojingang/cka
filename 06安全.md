## Authentication 认证

  验证用户身份是否合法

三种客户端访问API Server认证方式
├── 证书+私钥
├── Token
├── 用户名密码


## Authorization 授权

  对集群资源的访问控制，k8s授权处理以下的请求属性
- user,group,extra
- API,请求方法（如get,post,update,patch,delete）和请求路径(如/api)
- 请求资源和子资源
- Namespace
- API Group

  授权策略重点 RBAC
